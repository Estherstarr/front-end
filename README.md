Here is the backend repo [link](https://github.com/Estherstarr/hello-rails-backend) for the api that I created that displays data that is fetched in the front end.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running follow these simple example steps.

### Prerequisites

In order to run this project you need:

- Install ruby on your computer or run it with docker
- Have rails installed on your computer

### Setup

Clone this repository by using the command line :

- `https://github.com/Estherstarr/front-end.gitt`
- `cd front-end`

### Install

Install this project with:

`npm install`


### Usage


To run the project, execute the following command:

```sh
  npm run dev
```

- Then run the project on http://localhost:5173/, and you should see the greeting displayed as required.
